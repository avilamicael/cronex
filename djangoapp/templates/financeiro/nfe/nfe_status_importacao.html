Como funciona?
Busca Automática (a cada 4 horas)

Quando ativada, o sistema busca automaticamente novas notas fiscais na SEFAZ a cada 4 horas. As notas são salvas automaticamente no banco de dados para consulta posterior.

Busca Histórica (incremental)

Busca todo o histórico de notas disponíveis na SEFAZ de forma gradual (a cada 30 minutos). Útil para preencher notas antigas. Desativa automaticamente quando concluída.

Importante: A SEFAZ pode bloquear consultas excessivas (erro 656). O sistema trata isso automaticamente aguardando o próximo ciclo.
